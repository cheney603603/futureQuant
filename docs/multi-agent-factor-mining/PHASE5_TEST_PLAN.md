## 🧪 测试框架

### 使用 pytest

```bash
# 安装 pytest
pip install pytest pytest-cov pytest-xdist

# 运行所有测试
pytest tests/ -v

# 运行特定测试
pytest tests/unit/test_agent_base.py -v

# 运行并生成覆盖率报告
pytest tests/ --cov=futureQuant.agent --cov-report=html

# 并行运行测试
pytest tests/ -n auto
```

- **关键模块**: >= 95%
